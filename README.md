# ChatBot

## Description

This is a simple console-based chatbot application powered by the OpenAI GPT-3.5 API. It allows users to interact with a chatbot by typing messages in the console, and the chatbot responds with generated text.

## Features

- User-friendly console interface.
- Integration with the OpenAI GPT-3.5 API.
- Easy setup using a `.env` file for API key configuration.

## Prerequisites

Before running the application, ensure you have the following installed:

- Node.js
- npm (Node Package Manager)

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/chatbot-app.git
   cd chatbot-app

   ```

2. **Install dependencies:**

   ```bash
   npm install

   ```

3. **Create a .env file in the project root and add your OpenAI API key:**

   makefile
   OPENAI_API_KEY=your-api-key
   Replace your-api-key with your actual OpenAI API key.

## Usage

Run the application:

npm start
Interact with the chatbot by typing messages in the console.

To exit the application, type exit in the console.

## Configuration

The .env file is used to configure the OpenAI API key. Ensure it is present in the project root with the correct key.
